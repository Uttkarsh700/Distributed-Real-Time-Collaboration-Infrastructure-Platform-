# Frontend

The frontend will become the product experience for CloudCollab.

## Responsibilities

- Team dashboard and workspace navigation
- Monaco-based infrastructure and config editor
- Yjs-powered real-time collaboration UI
- Deployment logs and live status panels
- Metrics and activity views

## Current State

This repository stage only reserves the frontend folder and documents its role. No Next.js application is created yet, which keeps the foundation safe and avoids premature framework scaffolding.
