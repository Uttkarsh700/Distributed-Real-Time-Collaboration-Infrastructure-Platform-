# Features

## MVP Features

- User login foundation
- Workspace and project organization
- Shared project files for infrastructure and deployment configs
- FastAPI REST API foundation
- Deployment action placeholder
- Basic logs and metrics screens
- Documentation-first repository structure

## Advanced Prototype Features

- Real-time collaborative editing with Yjs
- Monaco Editor integration
- FastAPI WebSocket-based presence and live updates
- Deployment job orchestration through Celery and Redis
- Live log streaming into the UI
- Audit history for project changes and deploy events

## Future Production Features

- Role-based access control
- Multi-environment deployments such as dev, staging, and production
- Cloud provider integrations
- Approval workflows for deploys
- Searchable activity timelines
- Alerts, notifications, and incident management hooks
- Team-level analytics and utilization reporting
